# 微信传播导向重设计（V4）

## 核心目标

- 从战斗构筑玩法切换为“高密度事件抉择”的短局体验。
- 手机单手操作优先，保证首屏尽快看到可操作内容。
- 局末可一键复制分享文案并附带可点击游戏链接。

## 新玩法定义

- 名称：是男人就坚持100天（展示目标为挑战100天）。
- 核心循环：`事件 -> 三选一 -> 状态变化 -> 下一天`。
- 核心属性：现金、体力、心态、人设、热度。
- 失败条件：现金/体力/心态/人设任一归零。
- 结算：综合分 + 失败原因 + 整局剧情总结。

## 操作与人性化

- 关键按钮大面积、低误触。
- 每天主决策一次，降低认知负担。
- 临时技能每日刷新且限用一次，提供收益与代价取舍。
- 事件选项显示影响摘要，减少盲选挫败。

## 难度与可玩性策略

- 章节式因果线：事件由状态与标记驱动，不是纯随机。
- 延迟后果：部分决策在后续天数回收代价。
- 隐性压力：债务/过劳/舆论压力溢出会触发额外损耗。
- 反套路机制：同类策略重复使用会提高翻车概率。

## 微信传播机制

- 局末一键复制分享文案（分数、结局、复盘句、游戏链接）。
- 不再提供挑战码复刻功能。

## 技术落地

- 前端单页：`app/main.js + ui/game-ui.js + public/index.html`。
- 运行时文本状态：`window.render_game_to_text`。
- 结局故事：`POST /api/story/summary`（DeepSeek）。
