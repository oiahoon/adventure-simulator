## 对照清单

### 1) 紧凑移动端单卡滑动（文档 1294-1314）

- 状态：**已完成（基础版）**
- 已实现：
  - 单卡主区 + 左右滑阈值出牌
  - 半滑动左右预告高亮
  - 按钮兜底出牌
- 缺口：
  - 预告仍为文本，未做 icon 化影响提示
  - 手势阻尼/回弹细节仍可优化

### 2) 桌面手牌策略区（文档 1316-1324）

- 状态：**已完成（增强版）**
- 已实现：
  - mini hand 区
  - 可设为主卡
  - `hand` 变体下可直接左右出牌
  - `discard/defer` 手牌管理操作（含回收与冷却）
- 缺口：
  - hover 触发因子面板仍可增强（当前为轻量文本）

### 3) 混合模式（文档 1326-1333）

- 状态：**已完成（基础策略层）**
- 已实现：
  - 主卡 + 机会手牌共存
  - URL/按钮切换 `single|hand|auto`
  - 机会手牌支持 `弃置/延后`，延后牌后续回收
- 已新增：
  - 机会槽抽牌规则（`opportunitySlots`）
  - 机会偏好动作（`prefer: balanced/survival/growth/debt`）
  - UI 偏好切换控件（Web + CLI）
- 缺口：
  - 机会牌与主线牌视觉层级可再强化

### 4) 动效与反馈规范（文档 1337-1344）

- 状态：**大体完成**
- 已实现：
  - 滑动反馈/高亮
  - 主卡入场 + 文本延迟显隐
  - KPI 条动画（进度条宽度）
  - KPI 数值变化 pulse
  - 强制后续提示与红色标签
- 缺口：
  - 结算“单行反馈句”与更强的数值跳动动效仍可继续打磨

### 5) 结算复盘卡（文档 1374-1413）

- 状态：**已完成（增强版）**
- 已实现：
  - 关键牌 TopN
  - 高影响牌（按 impactScore）
  - 关键轨迹摘要（arc/forced/end）
  - 亮点摘要（策略触发 + 关键轨迹）
  - 分享图与分享文案同步复盘信息
- 缺口：
  - `memeTags` 仍未做结构化输出

### 6) A/B 指标与埋点（文档 1346-1360）

- 状态：**部分完成（可比较）**
- 已实现：
  - 会话指标 + 累计指标
  - variant 分桶（single/hand/auto）
  - 导出 JSON
  - `render_game_to_text` 输出 `ux_metrics`/`ux_analytics_totals`
- 缺口：
  - 未接入服务端/数据仓，当前仅本地 localStorage
  - 缺 Day1/Day7 留存口径
  - 缺卡牌熵/Gini 与 N 回合新鲜度更严谨计算
